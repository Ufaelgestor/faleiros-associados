Imagens do banner (hero):

1) Desktop: hero-banner.jpg
	- Formato: JPEG
	- Tamanho sugerido: 1600x900px (ou 1920x1080)
	- Peso alvo: < 500 KB

2) Mobile: hero-banner-mobile.webp (preferencial) ou hero-banner-mobile.jpg
	- Dimensão sugerida: 900x1200px (ou 1080x1350)
	- Peso alvo: < 250 KB

Dicas de otimização:
- Use WebP quando possível (melhor compressão).
- Comprima as imagens (Squoosh.app, TinyJPG, etc.).
- Evite arquivos acima de 1MB para manter o carregamento rápido.
